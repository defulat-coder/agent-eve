# Runtime-neutral Agent continuations

Agent Template exposes multi-turn state as an opaque Agent continuation plus structured Agent input responses. Runtime adapters own conversion to native state: Eve uses `SessionState` and `InputResponse`, while Claude persists its transcript, signs the session reference, acquires a single-use Redis lease, and uses the official deferred Tool resume flow. Shared schemas, API, and Web never expose Eve cursor fields, Claude session IDs, or deferred Tool IDs. This keeps the Agent runtime seam from ADR 0001 stable while allowing both runtimes to provide durable sessions and HITL end to end; Claude continuation replay or concurrent resume fails closed and a completed resume rotates the token.
